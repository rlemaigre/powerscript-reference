# Syntax

PixelsToUnits ( pixels, type )



- Table 10.578:




|Argument|Description|
|---|---|
|pixels|An integer whose value is the number of pixels you want to convert to PowerBuilder units.|
|type|A value of the ConvertType enumerated datatype value indicating how to convert the value:<br><br>• XPixelsToUnits! -- Convert the pixels in the horizontal direction.<br>• YPixelsToUnits! -- Convert the pixels in the vertical direction.<br>|
