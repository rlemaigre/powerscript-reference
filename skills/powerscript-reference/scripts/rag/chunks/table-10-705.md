Return value Integer. Returns 1 if it succeeds and -1 if an error occurs. Usage Use this function to indicate a range of preferred values. In a scheduling application, the selection range could indicate a block of time that is unavailable. Setting a selection range does not prevent the user from selecting a value either inside or outside the range. Examples This statement highlights the trackbar values between 30 and 70:



HTrackBar.SelectionRange( 30, 70 )
