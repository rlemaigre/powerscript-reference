# Examples

This statement prints the document in the RichTextEdit control rte_1. A Cancel dialog box displays to allow the user to cancel the printing:



rte_1.PrintEx(TRUE)

- 10.481 PrintGetPrinter Description Gets the current printer name. Syntax PrintGetPrinter ( ) Return value




String. Returns current printer information in a tab-delimited format: printername ~t drivername ~t port.
