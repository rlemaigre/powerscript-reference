# Syntax for JSONParser

objectname.ContainsKey ( ParentItemHandle, Key )



- Table 10.105:




|Argument|Description|
|---|---|
|objectname|The name of the JSONPackage or JSONParser object whose key you want to check.|
|ParentItemHandle|A long value specifying the parent item handle which is JsonObjectItem type.|
|Key|A string specifying the key name.|
