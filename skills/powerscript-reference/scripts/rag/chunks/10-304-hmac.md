# 10.304 HMAC

Calculates the HMAC value of a blob. Applies to CrypterObject object Syntax



crypter.HMAC ( algorithm, variable, key)



- Table 10.373:




|Argument|Description|
|---|---|
|crypter|The name of the CrypterObject object.|
|algorithm|A value of the HMACAlgorithm enumerated type that specifies the type of HMAC algorithm.<br><br>Values are:<br><br>• HMACMD5! – HMAC-MD5<br>• HMACSHA1! – HMAC-SHA1<br>• HMACSHA224! – HMAC-SHA224<br>• HMACSHA256! – HMAC-SHA256<br>• HMACSHA384! – HMAC-SHA384<br>• HMACSHA512! – HMAC-SHA512<br>|
|variable|A blob whose value is the data you want to process with HMAC.<br><br>When using the system blob function to convert a string to a blob, it is recommended to specify its encoding argument to be EncodingANSI! (for English characters only) or EncodingUTF8!, otherwise, the default EncodingUTF16LE! will be used.|
|key|A blob specifying the secret key.|
