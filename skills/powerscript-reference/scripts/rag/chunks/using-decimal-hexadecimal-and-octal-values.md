# Using decimal, hexadecimal, and octal values

Here is how to indicate a bullet (*) in a string by using the decimal, hexadecimal, and octal ASCII values:



- Table 1.4:




|Value|Description|
|---|---|
|"~249"|The ASCII character with decimal value 249|
|"~hF9"|The ASCII character with hexadecimal value F9|
|"~o371"|The ASCII character with octal value 371|
