# Table 10.765:

|Argument|Description|
|---|---|
|treeviewname|The name of the TreeView control in which you want to set the data for a specific item|
|itemhandle|The handle associated with the item you want to change|
|item|The TreeView item you want to change|



Return value Integer. Returns 1 if it succeeds and -1 if an error occurs. Usage



Typically, you would call GetItem first, edit the data, and then call SetItem to reflect your changes in the TreeView control.



Examples



This example uses the ItemExpanding event to change the picture index and selected picture index of the current TreeView item:



treeviewitem l_tvi long ll_tvi



ll_tvi = tv_list.FindItem(CurrentTreeItem! , 0) tv_list.GetItem(ll_tvi , l_tvi)



- l_tvi.PictureIndex = 5


- l_tvi.SelectedPictureIndex = 5 tv_list.SetItem( ll_tvi, l_tvi ) See also GetItem




10.616 SetJWTToken



Description



Sets the JWT token string to the HTTP request header which will be sent to the server in the following interface: Submit, SendGetRequest, SendPostRequest, SendDeleteRequest, SendPatchRequest, SendPutRequest, Retrieve, and RetrieveOne. If a token has been set in the HTTP request header, it will replace the original token.



Applies to RESTClient objects Syntax



objectname.SetJWTToken(string jwtToken)
