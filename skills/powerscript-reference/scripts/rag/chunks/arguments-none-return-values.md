Long. Return code choices (specify in a RETURN statement): 0 -- Continue processing Usage



When the user clicks in a horizontal scroll bar or presses the left arrow key on the keyboard in a horizontal track bar, nothing happens unless you have scripts that change the bar's Position property. For the scroll bar arrows and left arrow keys in a track bar, use the LineLeft and LineRight events; for clicks in the background above and below the thumb, use the PageLeft and Right events; for dragging the thumb itself, use the Moved event.
