# Table 10.348:

|Argument|Description|
|---|---|
|controlname|The name of the control for which you want to get today's date|



Return value Date. Usage



By default, the current system date is set as the Today date. You can use the SetToday function to specify a different date.



Examples This example retrieves the Today date in a DatePicker control into currentdate:



Date currentdate currentdate = dp_1.GetToday()



See also SetToday
