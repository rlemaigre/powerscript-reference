# Return value

Double. Returns the arctangent of n. Examples This statement returns 0:



ATan(0)



This statement returns 1.000 (rounded to three places):



ATan(1.55741)



This statement returns 1.267267 (rounded to six places):



ATan(Pi(1))



See also Tan ASin ACos ATan method for DataWindows in Section 2.4.6, “ATan” in DataWindow Reference.
