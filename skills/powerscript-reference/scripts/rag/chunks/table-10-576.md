|Argument|Description|

|hmenu|A long for the menu handle|
|nPos|An integer for the position of the menu item, counting from 0 at the leftmost or topmost position|
|caption|A string passed by reference that captures the name (Text property) of the menu item|
|nMaxLength|An integer that sets the maximum length of the value passed in the caption argument|



Return value



Long. Returns 1if it succeeds and -1 if an error occurs. If any argument is null, PBGetMenuString returns null.



Usage



Use PBGetMenuString to get the name of a menu item. This function is useful for some automated testing programs that cannot get menu item names from the Text property for menus that use the contemporary style.
