# Floating-point values

- Table 3.16: PowerBuilder datatypes for floating-point values




|Datatype in source code|Size, sign, precision|PowerBuilder datatype|
|---|---|---|
|float|32 bits, single precision|Real|
|double|64 bits, double precision|Double|



PowerBuilder does not support 80-bit doubles on Windows. Date and time



The PowerBuilder datatypes Date, DateTime, and Time are structures and have no direct equivalent for external functions in C.
