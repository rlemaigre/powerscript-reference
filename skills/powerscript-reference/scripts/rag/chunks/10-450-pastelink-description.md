# 10.450 PasteLink Description

Pastes a link to the contents of the clipboard into the control. The server application for the object on the clipboard must be running.



Applies to OLE controls Syntax



olecontrol.PasteLink ( )



- Table 10.571:




|Argument|Description|
|---|---|
|olecontrol|The name of the OLE control into which you want to paste the object on the clipboard|
