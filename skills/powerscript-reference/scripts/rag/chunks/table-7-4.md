# Table 7.4:

|Parameter|Description|
|---|---|
|testexpression|The expression on which you want to base the execution of the script|
|expressionlist|One of the following expressions:<br><br>• A single value<br>• A list of values separated by commas (such as 2, 4, 6, 8)<br>• A TO clause (such as 1 TO 30)<br>• IS followed by a relational operator and comparison value (such as IS>5)<br>• Any combination of the above with an implied OR between expressions (such as 1, 3, 5, 7, 9, 27 TO 33, IS >42)<br>|



|Parameter|Description|
|---|---|
|statementblock|The block of statements you want PowerBuilder to execute if the test expression matches the value in expressionlist|
