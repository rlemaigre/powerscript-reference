# 9.107 SizeCompleted Description

Occurs during the file compression or extraction process. This event can be used to display the total number of bytes in the file that has been compressed or decompressed.



Event ID



Table 9.239:



|Event ID|Objects|
|---|---|
|None|CompressorObject and ExtractorObject|



Arguments



Table 9.240:



|Argument|Description|
|---|---|
|CompletedSize|A longlong whose value is the total number of bytes in the file that has been compressed or decompressed.|



Return Values None See also Error Finished ProcessingFile Start
