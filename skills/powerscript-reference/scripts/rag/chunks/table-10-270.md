# Table 10.270:

|Argument|Description|
|---|---|
|objectname|A reference to the TokenRequest object in which you want to get the request header.|



Return value String. Returns the header information of all requests. If any argument's value is null, the method returns null. Examples



The following example shows the use of the GetHeaders function to get the value of all headers:



string ls_headers TokenRequest lnv_TokenRequest



ls_headers = lnv_TokenRequest.getheaders()



See also AppendParam ClearHeaders GetHeader SetHeader SetHeaders
