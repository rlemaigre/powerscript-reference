# 10.530 Resize Description

Resizes an object or control by setting its Width and Height properties and then redraws the object.



Applies to Any object, except a child DataWindow Syntax



objectname.Resize ( width, height )



- Table 10.656:




|Argument|Description|
|---|---|
|objectname|The name of the object or control you want to resize|
|width|The new width in PowerBuilder units|
|height|The new height in PowerBuilder units|
